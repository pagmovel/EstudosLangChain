# Arquivos Complementares do Projeto LangChain

Este pacote contém arquivos grandes necessários para o projeto LangChain que são distribuídos separadamente do repositório principal.

## Arquivos Incluídos

### Ferramentas (pasta tools_project/)
- ffmpeg.exe - Ferramenta para processamento de áudio e vídeo
- ffprobe.exe - Ferramenta para análise de mídia
- langchain.zip - Pacote com recursos adicionais do LangChain
- rustup-init.exe - Instalador do Rust
- vs_BuildTools.exe - Visual Studio Build Tools

### Documentação (pasta files/)
- LLM.pdf - Documentação sobre Large Language Models
- apostila.pdf - Material de referência do curso

## Instruções de Instalação

1. Baixe e extraia este pacote
2. Copie os arquivos .exe e .zip para a pasta 'tools_project' do seu projeto
3. Copie os arquivos .pdf para a pasta 'files' do seu projeto

Nota: Certifique-se de que a estrutura de pastas corresponda à do repositório principal.
